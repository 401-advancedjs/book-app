### Setup
#### `.env` requirements
* `PORT` - Port Number
* `MONGODB_URI` - URL to the running mongo instance/db
* `DATABASE_URL` - URL to the running postgres instance/db
* `ACTIVE_DB` - mongo for mongodb, or blank for postgree db
